株式投資を始めると、「毎月分配」「年率10%超」といった高い分配金をうたうETF（上場投資信託）が目に入ります。JEPI、JEPQ、QYLDといった名前を聞いたことがあるかもしれません。銀行預金ではなかなか大きな利息がつかないなか、年10%が毎月配られるとしたら、とても魅力的に見えます。

でも、そのお金はどこから来ているのでしょうか。この記事では、その正体を、専門用語をできるだけ使わずに解きほぐしていきます。結論を先に言うと、高分配ETFの多くは、将来得られるかもしれない値上がり益の一部を、いまの現金収入に変換している商品です。お金を無から生み出しているわけではありません。個人的には、この一点を最初に腹落ちさせておくと、あとの話がずっと分かりやすくなると思っています。

## なぜ株の配当以上のお金を配れるのか

普通の株の配当は、多くの場合で年に数%です。高配当で知られる銘柄でも、4%や5%あれば高いほうです。ところが、高分配ETFのなかには、年率10%前後の分配金を毎月配るものがあります。

この差はどこから来るのでしょうか。答えは、分配金の「原資」、つまり配るお金のもとが違うところにあります。普通の高配当株は、企業が稼いだ利益の一部を配当として出します。一方で高分配ETFの多くは、株式の配当に加えて、オプションという仕組みを使って受け取ったお金も分配に回しています。

この「オプションを使った収入」が、高い分配金のからくりです。順を追って見ていきます。

## 普通の高配当株と、高分配ETFは別物

まず、両者は名前が似ていても中身が別物だと押さえておきます。

| | 普通の高配当株 | オプションを使う高分配ETF |
| --- | --- | --- |
| 分配（配当）のもと | 企業が稼いだ利益 | 株式の配当 ＋ オプションを売った収入など |
| 分配の仕組み | 企業が利益から支払う | ETFがオプション取引で受け取った代金を配る |
| 値上がり益 | そのまま受け取れる | 大きな上昇の一部を手放していることがある |

高配当株を集めただけのETFもありますが、年率10%前後の分配をうたう商品の多くは、後者のオプションを使うタイプです。同じ「高い分配金」でも、成り立ちが違います。

## オプションは「権利の売買」

オプションという言葉が出てきたので、ここだけ説明します。むずかしく考える必要はありません。オプションとは、将来、決めた価格で売買できる「権利」のことです。

権利なので、売り買いができます。

- 権利を買う側は、その対価としてプレミアムというお金を払います
- 権利を売る側は、そのプレミアムを受け取ります

保険にたとえると分かりやすいかもしれません。保険を買う人は保険料を払い、保険を売る会社は保険料を受け取ります。オプションを売る側は、この「保険を売る会社」に近い立場です。プレミアムという料金を先に受け取る代わりに、相手が権利を使ってきたら、それに応じる義務を負います。ただし、保険と完全に同じ仕組みではないので、あくまでイメージとして受け取ってください。

高分配ETFは、このオプションを「売る側」にまわって、プレミアムを受け取っています。この記事で覚えておく専門用語は、オプション、プレミアム、そして次に出てくるコールオプションとカバードコールだけです。

## カバードコールで分配金が生まれる流れ

高分配ETFがよく使うのが、カバードコールという方法です。これは、株を持ったまま、その株のコールオプションを売るやり方です。コールオプションとは「将来、決めた価格で買う権利」のこと。それを売る、というわけです。

具体的な数字で追ってみます。ここで出す金額は、仕組みを説明するための仮の例です。

1. いま、100万円ぶんのNASDAQ100（米国のナスダック市場に上場する大手企業で構成される株価指数）を持っているとします
2. 1か月後に「105万円で買える権利」を、誰かに売ります（権利行使価格をいくらに置くかは、商品によって異なります）
3. その対価として、たとえば2万円のプレミアムを受け取ります
4. ETFは、この2万円などを分配金のもとにできます

ここまでは、じっとしていてもお金が入ってくるように見えます。でも、代償があります。もし1か月後にNASDAQ100が120万円まで上がっても、あなたは「105万円で売る」約束をしているので、105万円を超えた上昇のぶんは十分に受け取れません。上がってほしい場面で、上値が頭打ちになるのです。

つまり、こういうことです。

> お金を無から生み出しているのではなく、将来の上昇余地を、現在の収入に変えている。

受け取った2万円は、じつは「大きく上がったときの利益」と引き換えに得たものだった、というわけです。

## 相場が動くと、何が起きるか

この戦略は、相場の動き方によって結果が変わります。4つのパターンで整理します。

| 相場 | 何が起きるか |
| --- | --- |
| 横ばい | 株価はほとんど上がらないが、受け取ったプレミアムが収益になる。得意な場面 |
| 緩やかな上昇 | 決めた価格までの値上がり益と、プレミアムの両方を狙える |
| 急上昇 | 上昇余地を売っているので、普通の指数ETFに置いていかれやすい |
| 下落 | プレミアムが損失のクッションになるが、下落を完全には防げない |

高分配ETFが心地よく見えるのは、横ばいや緩やかな上昇のときです。逆に、相場が大きく上がる局面では、普通に指数を持っているほうが有利になりがちです。プレミアムという定期収入と引き換えに、大きな上昇に乗る力を手放しているからです。

## QYLDを例に見てみる

代表的な商品で確認します。QYLD（Global X Nasdaq 100 Covered Call ETF）は、NASDAQ100に投資しながら、その指数のコールオプションを売り、受け取ったお金などを毎月分配するETFです。まさに、ここまで説明してきたカバードコールを、そのまま商品にしたものです。

https://www.globalxetfs.com/funds/qyld/

分配利回り（投資額に対して、1年でどれだけ分配金が出るかの割合）は高く、公式の情報では、この記事を書いている2026年7月の時点で、直近1年の分配は概ね年率10%台前半でした。ただし、この数字は相場やオプションの状況で変わります。最新の値は必ず公式の情報で確認してください。

そしてQYLDも、これまでの説明どおり、NASDAQ100が大きく上昇する局面では、指数そのものにはなかなか追いつけません。高い分配金と、大きな上昇に乗りにくいことは、表裏の関係にあります。なお、JEPIやJEPQも似た発想の商品ですが、何に投資し、どこまで上昇余地を残すかは少しずつ違います。三つが同じ中身というわけではありません。その違いをどう見るかは、次回に譲ります。

## 分配利回り10%なら、年10%儲かるのか

ここが一番の落とし穴です。私自身、投資を始めたころにいちばん誤解していたのがこの点でした。「分配利回り10%」という表示を見ると、預金の利息のように、資産が毎年10%ずつ増えていくように感じます。ですが、それは正しくありません。

簡単な例で考えます。これも説明のための仮の数字です。

- 100万円ぶんのETFを買いました
- 仮に、1年間で分配金を10万円受け取ったとします（分配利回り10%にあたります）
- ところが同じ1年で、そのETFの価格は100万円ぶんから90万円ぶんに下がっていました

このとき、あなたの手元はどうなっているでしょうか。分配金でプラス10万円、値下がりでマイナス10万円。合わせると、増えても減ってもいない、ほぼゼロです。

| 項目 | 金額 |
| --- | --- |
| 受け取った分配金 | ＋10万円 |
| ETF価格の値下がり | −10万円 |
| 合計（トータルリターン） | ほぼ0円 |

大事なのは、分配利回りという一つの数字ではなく、価格の変動と分配金を合わせた成績、つまりトータルリターンで見ることです。分配金が高くても、その裏で価格が下がっていれば、資産は増えていないのです。

さらに、高分配ETFの分配金には、運用で得た利益だけでなく、実質的に自分が預けた元本の一部を払い戻しているものが含まれることもあります。毎月お金が入るのに、なぜか資産が増えないと感じる背景には、こうした事情があります。この点は、連載の第3回でくわしく扱います。

## どんな人に向くのか、そしてまとめ

ここまでを踏まえると、高分配ETFが合うかどうかは、その人が投資に何を求めるかで変わります。次の表は「こういう人が買うべき」という助言ではなく、仕組みと自分の目的が合っているかを考えるための材料です。実際に合うかどうかは、資金の状況や税金、他の資産との組み合わせなどによっても変わります。

| 仕組みと目的が合いやすい考え方 | いったん立ち止まったほうがよい考え方 |
| --- | --- |
| 毎月のキャッシュフロー（定期的な入金）を重視している | 長い目での資産の最大化を最優先している |
| 大きな値上がりより、定期収入のほうがほしい | 分配利回りの数字だけで商品を選んでいる |
| 上値が抑えられる仕組みだと理解して持てる | 価格が下がる可能性を理解していない |
| 分配金は利益と元本払戻が混ざりうると分かっている | 銀行預金の利息のようなものだと思っている |

この記事の要点をまとめます。

- 高分配ETFの分配金は、株式の配当だけでなく、オプションを売った収入などから生まれています
- そのオプション収入には、大きな上昇に乗る力を手放す、という代償があります
- 分配利回りの高さではなく、価格変動と分配金を合わせたトータルリターンで判断する必要があります

高分配ETFは、良い悪いで割り切れる商品ではありません。将来の値上がり益を、いまの現金収入に付け替えている。その仕組みと代償を理解したうえでなら、目的に合う人には選択肢になります。分配金の数字だけで飛びつかない。それが、この連載を通してお伝えしたいことです。

なお、この「カバードコールで上昇を手放す」という話は、ビットコインのオプションを題材に、実際のデータで検証した連載でも詳しく扱いました。数字で確かめたい方は、あわせてどうぞ。
