## 個別ニュースに見えて、根っこは「1つの再分類」

2026年、日本の暗号資産まわりのニュースが急に増えた。インサイダー取引規制の創設、約20%の分離課税、交換業者への業規制強化、取扱銘柄の情報開示。バラバラの話題に見える。

だが、これらの多くは1つの大きな方向にひもづいている。「暗号資産を、決済法制中心の枠組みだけでなく、"投資商品"としても規律し直す」という流れだ。具体的には、投資・取引に関する規制を金融商品取引法（金商法）へ移し、税制もそれに連動して見直される（[Bloomberg](https://www.bloomberg.co.jp/news/articles/2026-06-11/TGGBNTT96OSH00)、[ビットタイムズ](https://bittimes.net/news/220515.html)）。規制の改正と税制の改正は別々の立法だが、「投資商品として扱う」という同じ方向を向いている。

この記事は、個別ニュースを1つずつ追うのではなく、その方向1本を軸に読む。なぜ決済法だけでは足りず金商法なのか、そこから何が波及するのか、そして私がこれまで書いてきたSBIの取引所再編やステーブルコイン、STの動きが、なぜここで地続きに見えるのかを整理する。法律の細かい条文には踏み込まない。

## 「決済手段」から「金融商品」へ、2つの法律の思想

まず、2つの法律の性格が違う。ここが起点になる。

資金決済法は、支払い・決済の手段を規律する法律だ。決済サービスの健全な発展や利用者保護、決済システムの安全性に軸足がある。暗号資産についても、交換業者の登録制や利用者財産の分別管理といった規制は、もともとここに置かれていた。一方の金商法は、株式や債券のような投資性の商品を規律し、投資家保護に軸足がある。継続的な情報開示、不公正取引の規制、業者への業規制。投資対象を扱うための仕組みが揃っている。

暗号資産は当初、「決済手段」として資金決済法の下に置かれた。ところが実態は、支払いよりも投資対象としての売買・保有が広がっていく。利用者保護の規制自体は資金決済法にもあったが、発行者側の情報開示や、未公表情報を使った取引への規制といった、投資商品としての規律には空白があった（[第一生命経済研究所](https://www.dlri.co.jp/report/macro/597970.html)、[金融庁 説明資料](https://www.fsa.go.jp/common/diet/221/02/03.pdf)）。

そこで、暗号資産の投資・取引に関する規制を金商法へ移し、暗号資産の特性に応じて規律し直す方向が示された。ここで注意したいのは、これは暗号資産を株式や社債のような有価証券そのものにするのではなく、金商法上の規制対象として投資・取引のルールを整える設計だという点だ。補助線を1本引くと分かりやすい。ST（セキュリティトークン）は、その裏付けとなる権利が金商法上の有価証券に当たるため、2019年改正・2020年5月施行の金商法で既に規制対象になっている。暗号資産の移管はそれとは別の枠組みで、投資性のある領域に金商法型の規律を広げる動きだ。

## この転換から、何が波及するか

位置づけが「決済手段」から「投資商品」へ動くと、影響は4つの方向に広がる。ニュースで見た個別の話は、だいたいこのどれかに収まる。

| 方向 | 何が変わるか |
| --- | --- |
| 規制 | 交換業者は金商法上の業規制へ（名称も見直しの方向）。インサイダー取引規制の創設、発行者等による情報開示の整備、不公正取引規制の強化 |
| 税制 | 原則は雑所得の総合課税（高所得者ほど高く最大約55%）から、一定の取引に約20%の申告分離課税へ（見込み）。3年の繰越控除も |
| 事業者 | 業規制が重くなり、参入と事業維持のハードルが上がる。追加コストが体力のない業者への再編圧力になり得る |
| 個人 | 投資家保護（開示・不公正取引規制）が及ぶ。税負担は所得水準によっては軽くなる。未公表情報を使った取引の禁止などは個人にも及ぶ |

規制面では、ビットコインやイーサリアムそのものが有価証券になるわけではないが、取扱業者や不公正取引が金商法の枠組みに入る。株式のインサイダー取引規制を参考に、暗号資産の未公表の重要情報を使った取引を規制する枠組みも導入される（[長島・大野・常松法律事務所](https://www.nagashima.com/publications/publication20260424-1/)）。ただし発行主体のない暗号資産も多く、対象者や重要事実の立て方は株式とそのまま同じにはできない。従来の資金決済法上の規制に、開示や不公正取引といった金商法型の規律が加わる、と捉えるのが正確だ。

税制の変化は個人に効く。現行では暗号資産の利益は原則として雑所得の総合課税で、他の所得と合算されるため高所得者ほど税率が上がる。住民税を含めた高所得部分の限界税率は概算で約55%になり得る（利益全体に一律55%がかかるという意味ではない）。これが、一定の暗号資産取引について約20%（復興特別所得税を含めて20.315%）の申告分離課税へ変わる方向だ。2026年3月31日に成立した税制改正法にこの分離課税が盛り込まれているが、その適用は金商法側の改正の施行などを条件としている（[CoinDesk JAPAN](https://www.coindeskjapan.com/329002/)）。金商法へ移れば自動的に20%になるわけではなく、税制は別の立法で決まる。損失の繰越控除（3年）も同じ区分内での話で、本稿執筆時点の公表資料からは、上場株式などとの横断的な損益通算まで認められるとは確認できない。

## 連載の「なぜ」が、ここで回収される

ここが、この記事を書きたかった理由だ。私はこれまで、SBIによる交換業者の取得・再編、円ステーブルコインの発行スキーム、不動産ST、国債STを別々に書いてきた。それぞれ面白いのだが、「投資性のあるトークン化資産をどう規律するか」という同じ土俵の上に乗っている。

SBIが相場の弱い局面でbitbankなどの取得に動いた狙いは、顧客基盤や取引インフラの獲得など複数が考えられる。そのうえで、業規制が重くなる局面では登録済み・規制対応済みの事業者の価値が上がるので、この制度変更は今後の再編を後押しする要因になり得る。SBI自身が金商法移管を買収理由に挙げているわけではない点は断っておく。

ステーブルコインとの関係も、この軸で整理できる。法定通貨建てで額面償還される一定のステーブルコインは、電子決済手段として資金決済法の側で規律される。一方、投資性のある暗号資産は金商法側の規律へ動く。同じブロックチェーン上のトークンでも、発行・償還の構造や表章する権利によって行き先が変わる。棲み分けの線が、ここで見えてくる。

そしてSTは、その権利が有価証券に当たるため、以前から金商法の枠内にある。今回の暗号資産移管とあわせると、投資性のあるトークンについては金商法の傘が広がっていく。個人的に一番の見どころはここだ。バラバラに見えた動きが、「投資性のあるトークンには金商法の規律が広がる」という一本の線で見通せてくる。もっとも、金商法に入るかどうかは名称ではなく、そのトークンが表章する権利や販売の構造で個別に決まる。一般的なNFTや電子マネー、トークン化預金など、この傘に一律には入らないものもある。

## まだ「移行中」であることに注意

期待だけで語らないよう、時制を正確にしておきたい。以下は2026年7月13日時点の状況で、これはまだ「決まりつつある」段階だ。法案の成立・公布・施行・適用開始は、それぞれ別のタイミングになる。

| 時期 | できごと |
| --- | --- |
| 2025年12月 | 金融審議会ワーキング・グループ報告 |
| 2026年3月31日 | 所得税法等の改正が成立（分離課税は金商法側改正の施行等が前提） |
| 2026年4月10日 | 改正法案（金商法・資金決済法の一部改正）を国会に提出 |
| 2026年6月11日 | 衆議院で可決、参議院へ |
| 2027年中（見込み） | 施行の見通し |
| 2028年1月（見込み） | 一定の暗号資産取引の譲渡に約20%分離課税の適用見込み |

つまり、税制改正法そのものは2026年3月に成立しているが、分離課税の適用は金商法側の改正の施行などを条件にしており、いまの取引にすぐ効くわけではない。インサイダー規制など金商法改正の内容も、施行されて初めて動く。ニュースの見出しだけを見て「もう20%課税になった」と早合点すると、時期を読み違える。最新の審議・成立状況は一次情報で確認してほしい。

## まとめ

- 2026年に飛び交う暗号資産のニュース（インサイダー・分離課税・業規制・開示）の多くは、「暗号資産を投資商品として規律し直す」という同じ方向にひもづく
- 資金決済法は決済サービスと利用者保護、金商法は投資家保護（開示・不公正取引・業規制）。実態が投資対象化したため、投資・取引の規制を金商法へ移す。ただし暗号資産が有価証券そのものになるわけではない
- 波及は規制・税制・事業者・個人の4方向。個人は一定の取引が最大約55%の総合課税から約20%の申告分離課税へ（施行後の見込み。税制は別の立法で決まる）
- この軸で見ると、SBIの取引所再編・ステーブルコインの棲み分け・STの位置づけが地続きに見える。投資性のあるトークンに金商法の規律が広がる（一般的なNFT等は名称でなく権利構造で個別判断）
- 情報は2026年7月13日時点。法案は衆院可決の段階で、施行は2027年、分離課税は2028年の見込み。確定として書かない

個別ニュースを1つずつ追うと、暗号資産まわりは分かりにくい。だが「決済手段から投資商品へ」という1本の軸を引くと、私にはこれまでバラバラだった話が地続きに見えた。2026年は、その方向づけが形になっていく年だ。次の見どころは、実際の施行と、業規制・分離課税の細部がどう固まるかになる。
