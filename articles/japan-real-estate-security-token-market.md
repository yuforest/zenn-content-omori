## 静かに1兆円に届こうとしている市場

派手な暗号資産の値動きが報じられる裏で、地味だが本気の市場が立ち上がっている。ブロックチェーン上で発行されるデジタル証券、いわゆるセキュリティトークン（ST）だ。

国内のST市場は拡大を続けている。野村の総括レポートによれば、公募STの累計発行額は2025年度で3,000億円を超え、2026年度には5,000億円規模に達する見込みだ（[NOMURA 市場総括レポート](https://www.nomuraholdings.com/jp/news/nr/bstr20260402.html)）。集計範囲を広げた市場規模では、2025年末で約5,831億円、2026年中に1兆円規模になると予測する見方もある（[三井住友トラスト基礎研究所](https://www.smtri.jp/report_column/info_cafe/2025_03_17_6501.html)）。数え方によって幅はあるが、市場が1兆円に手が届くところまで来ている点と、その大半が不動産を裏付けとするもの（公募STの約85%）である点は、どのレポートでも共通している。

海外のトークン化（RWA）と比べると、毛色の違いが際立つ。海外では米国債など金融資産のトークン化が市場拡大を牽引しており、不動産だけでなくTreasuryやCreditまで含めた市場は2026年時点で約290億ドル規模、BlackRockなどが主役だ。同じ現実資産のトークン化でも、牽引役が違う。海外は金融資産が中心、日本は不動産が中心なのだ。この記事では、なぜ日本で不動産のトークン化が先行したのか、REITと何が違うのか、そして「そもそもSTにする必要があるのか」という当然の懐疑まで、市場と構造の視点で見ていく。実装の細部やコードには踏み込まない。

## そもそも不動産STとは（30秒で）

最小限だけ押さえる。不動産STは、ブロックチェーン上で発行されるデジタル証券で、単一の不動産を裏付けに、その賃料収入などを投資家に分配する商品だ。譲渡には制限が付き、機関投資家から個人まで販売される。

ざっくり言えば「1棟のビルやマンションを、デジタルの証券にして小口で持てるようにしたもの」と思えばいい。ここから先の「なぜ不動産なのか」が本題だ。

## なぜ「不動産から」始まったのか

日本のトークン化が不動産から立ち上がったのには、いくつかの構造的な理由がある。

まず、法的なルートが既に整っていた。ここが他ではあまり語られない核心だ。日本では、不動産を信託に入れて「受益証券発行信託（信託受益権）」の形にする枠組みが、信託法・金融商品取引法のもとで先に確立していた。

```
不動産 → 信託受益権 → セキュリティトークン化
```

この「不動産→信託受益権→ST」という道筋が、制度としてもっとも整備されていたため、債券や株式よりも不動産が最初の受け皿になった。新しい商品を一から法制度ごと作るのではなく、既存の信託の枠組みにトークンを載せられたことが、立ち上がりの速さを生んだ。

そのうえで、裏付け資産が分かりやすい。不動産は現物があり、価格も鑑定評価をベースに説明できる。「何を裏付けにした証券なのか」が明確で、安定した商品として語りやすい。値動きの理屈が曖昧になりがちな金融商品より、最初の一歩として扱いやすかった。

次に、訴求が直感的だ。これまでタワーマンションや大型オフィスといった大型物件は、実質的に機関投資家しか買えなかった。それを個人が小口で持てる、という手触りのある投資は、説明が要らないほど分かりやすい。

そして、既存プレイヤーが乗りやすかった。大手の信託銀行・証券会社・デベロッパーは、もともと不動産と証券の両方に強い。彼らの持ち場の延長線上に不動産STがあったため、座組みが組みやすかった。

最後に、商品設計が日本市場に合っていた。不動産STは、短期で価格変動を狙う投機商品ではなく、中長期で賃料収入を得る資産形成商品として設計されている。ここが、値上がり期待で語られがちなクリプトと対照的で、堅実志向の日本の個人投資家に馴染んだ。

## REITと何が違うのか

読者が一番気にするのはここだろう。「それ、REITと何が違うの?」という疑問だ。表で整理する。

| 観点 | J-REIT | 不動産ST |
| --- | --- | --- |
| 分散 | 複数物件で分散が効く | 単一物件が多い（複数物件型・ポートフォリオ型もある） |
| 流動性 | 市場で日々売買できる | 証券会社の買取や限定的な二次流通が中心 |
| 価格 | 市場で常に変動（株式相場の影響を受けやすい） | 鑑定評価が価格の重要な参考になり、現状は値動きが穏やか |
| 立ち位置 | 上場した不動産小口化商品 | 現物でもREITでもない「第3の選択肢」 |

要点は、REITが「株式のように売買できる不動産の詰め合わせ」なのに対し、不動産STは「特定の物件を、現状は値動き穏やかに中長期で持つ」商品だという点だ。株式相場に連動して上下するREITとは、狙っている投資体験がそもそも違う。ただし、二次流通が活発になれば不動産STも価格変動は大きくなり得る。あくまで現状の話だ。だから不動産STは、現物不動産でもREITでもない第3の選択肢として位置づけられている（[NOMURA ウェルスタイル](https://www.nomura.co.jp/wealthstyle/article/0633/)）。

## なぜ三菱UFJ / Progmat が主導するのか

日本の不動産STは、少数の大手が主導する「上から」型で立ち上がった。なかでも、三菱UFJ信託銀行が主導するデジタル証券基盤 Progmat が、累計発行額・案件数で国内最大級の実績を持つ。もっとも、国内には野村系のBOOSTRYや、ibet、Securitize Japanといった他の基盤もあり、Progmatが独占しているわけではない。

大型案件を見ると、その規模感が分かる。

- 汐留シティセンター：KJRM・三菱UFJ信託銀行・Progmatによる案件で、公募発行額は約314億円。国内最大規模の不動産STとされる（[CoinDesk Japan](https://www.coindeskjapan.com/317046/)）。
- MUFGリアルティ・トークン 大阪堂島浜タワー：発行額224億円。MUFGグループが一体で手がけた案件としては初とされる（[野村](https://www.nomuraholdings.com/jp/news/nr/nsc20260304.html)）。

なぜ信託銀行を核にした座組みが、大手への集中とセットになるのか。理由は、この商品が「信頼」で成り立っているからだ。信託受益権の管理、権利関係を記録する原簿の運営、そして投資家の資産を発行体の倒産から守る倒産隔離。これらを信託銀行という枠組みで担保している。裏を返せば、この座組みを組めるプレイヤーは限られる。だから自然と大手金融に集中する。この「信頼の構造」は、後半で扱うリスクの話に直結する。

## 流動性のボトルネックと二次流通（ODX / START）

不動産STの弱点は、はっきりしている。「売りたいときに売りにくい」ことだ。単一物件で、基本は買取や限定的な流通に頼るため、REITのような日々の売買はできない。

この弱点を埋める仕組みとして立ち上がったのが、二次流通市場だ。SBIグループの大阪デジタルエクスチェンジ（ODX）が、2023年12月に国内初のST二次流通市場「START」を、私設取引システム（PTS）として開業した（[ODX START](https://www.odx.co.jp/st/)、[日経クロステック](https://xtech.nikkei.com/atcl/nxt/column/18/00001/08988/)）。

ただし、冷静に見ておきたい。現状のSTの二次流通は、乱立する銘柄間のスイッチングや、必要なときの償還が中心で、株式のように活発に売買される段階にはまだ達していない。流動性は発展途上、というのが正直なところだ。この流通インフラをどう厚くするかは、SBI経済圏の動きとも重なる論点で、姉妹記事で別途扱っている。

## 「そもそもSTにする必要ある?」という当然の疑問に向き合う

ここからが後半の核だ。不動産STを持ち上げるだけでは、読み物として片手落ちになる。当然湧いてくる懐疑を、賛否両論で正面から扱う。

### ①「ブロックチェーンである必要があるのか」問題

もっともな疑問だ。小口の不動産投資なら、従来の不動産小口化商品も、REITも、不動産クラウドファンディングもある。わざわざブロックチェーンに載せる技術的な必然性は、本当にあるのか。

一歩踏み込むと、STの意味は「証券そのもの」より「基盤」にある。発行・原簿（権利の記録）・譲渡・決済を、一つの共通基盤の上でまとめて処理できる。従来はバラバラの事業者や台帳に分かれていた工程を、同じインフラに載せられる点にこそ効率化の芽がある。加えて、将来のプログラマビリティ（配当の自動化や、二次流通での即時決済）も見込める。

ただ正直に言えば、これらの多くはまだ「未来への布石」の側面が強い。今この瞬間に、STでなければ実現できない、と言い切れる段階ではない。ここは誇張せず認めておくのがフェアだ。

### ②「信頼できるのか」問題（信用リスクとオペレーショナルリスク）

リスクは種類を分けて捉えると見通しがいい。まず信用リスクは、原資産（不動産そのものの価値や賃料）、発行体や保証会社の財務、そして信託受託者にひもづく。ここが本丸だ。これとは別に、ブロックチェーン基盤やプラットフォーム運営の不確実性で受け渡し・配当・償還が遅れるオペレーショナルリスク、そしてサイバーセキュリティのリスクがある（[日本総研](https://www.jri.co.jp/page.jsp?id=103560)）。基盤の話は信用リスクそのものではなく、運用面のリスクとして分けて見るのが正確だ。

一方で、日本のSTはこれらに無防備なわけではない。前述のとおり、信託銀行を核にした座組みで、倒産隔離や資産保全に配慮した設計になっている。「だから大手金融が主導しているのだ」という第5節の話が、ここで効いてくる。リスクがゼロではないが、それを抑える構造は組み込まれている、という理解が正確だ。

### ③「流動性が低い」問題

これは弱点として認めるほかない。単一物件で分散が効かず、銘柄数もまだ少なく、二次流通も薄い。しかも元本保証はない。売りたいときに希望価格で売れる保証はない、という前提で持つ商品だ。

ただし、これも現在進行形の論点だ。PTS（私設取引システム）の活性化や、STARTのような二次流通の拡充で、流動性は改善が期待されている。今の薄さを固定的に捉えるか、伸びしろと捉えるかで、評価は変わる。

### ④ 法・規制の未成熟

STは金融商品取引法などの枠組みで扱われるが、流通市場まで見据えた制度整備は、まだ発展途上だ（[PwC Japan](https://www.pwc.com/jp/ja/knowledge/prmagazine/pwcs-view/202303/43-06.html)）。ルールが固まりきっていない領域である点は、リスクとして頭に置いておきたい。

## で、結局STは「買い」なのか

過度な期待も、過度な悲観もしないで着地したい。

不動産STは、値上がりを狙う投機商品ではない。「大手が開発した大型物件を、個人が中長期で穏やかに持てる第3の選択肢」というのが、本来の位置づけだ。ここを取り違えて、クリプト的な短期リターンを期待すると、まず失望する。

そのうえで、信託銀行・証券会社・デベロッパーが継続的に大型案件を組成していること自体が、市場への本気度を示している。腰を据えたプレイヤーが、実際に数百億円規模の物件を次々とトークン化している。ただし、ブロックチェーンである技術的な必然性と、流動性の薄さは、まだ宿題として残っている。この2つが埋まっていくかどうかが、次の見どころになる。

視野を広げると、海外のRWAは米国債など金融資産のトークン化が牽引し、BlackRockなどが主役だ。日本は不動産が中心で、静かに、しかし着実に伸びている。同じトークン化でも、牽引役はこれほど違う。どちらが正解というより、それぞれの市場の事情が、そのまま出発点の違いになっている。

日本のトークン化金融をめぐっては、SBIの経済圏や円ステーブルコインの発行スキームについても書いている。あわせて読むと、不動産STがその大きな流れのどこに位置するかが見えてくるはずだ。
